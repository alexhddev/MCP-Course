# Steps for MCP Python project

1. **Initialize the project in emtpy folder:**
   ```bash
   uv init
   ```

2. **Install MCP dependencies:**
   ```bash
   uv add "mcp[cli]"
   ```

3. **Create HelloName mcp files:**
   - create server.py
   - create main_stdio.py

4. **Test the MCP:**
   - create .vscode/mcp.json file and add the server
